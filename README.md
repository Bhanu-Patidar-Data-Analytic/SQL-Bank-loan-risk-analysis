# Bank Loan Risk Analysis – SQL

## 📌 Objective
Analyze loan approval and default trends to identify high-risk customer segments.

## 🛠 Tools Used
- MySQL
- SQL (JOIN, GROUP BY, HAVING, CASE)

## 📊 Analysis Performed
- Loan approval rate analysis
- Default trend analysis
- Risk segmentation based on customer attributes
- Aggregated financial reporting using SQL

## 📈 Key Insights
- Identified high-risk borrower profiles
- Analyzed approval patterns across categories
- Generated structured reports to support lending decisions
